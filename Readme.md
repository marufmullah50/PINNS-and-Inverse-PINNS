# Numerical Methods and Physics-Informed Neural Networks (PINNs)

This repository contains implementations of **numerical methods** and **Physics-Informed Neural Networks (PINNs)** for solving **1D and 2D differential equations** arising from **engineering, materials science, and multiphysics problems**.  
The main focus is on comparing **classical numerical solvers** with **deep learning–based physics-informed approaches**.

---

## Contents

### 1. Numerical Solutions
- Python-based numerical solvers  
- 1D and 2D mesh generation  
- Finite Difference / Finite Element–style discretizations  
- Solutions of governing equations on structured meshes  

---

### 2. Physics-Informed Neural Networks (PINNs)
- PINNs for **1D and 2D forward problems**
- Implemented using **PyTorch**
- Benchmark problems solved using **DeepXDE**
- Comparison with numerical solutions  

---

### 3. Inverse PINNs
- Parameter identification from sparse or noisy data  
- Learning unknown material properties and physical coefficients  
- Hybrid physics–data-driven modeling  
- Inverse problems in engineering and materials systems  

---

### 4. Materials and Multiphysics Problems
- Heat conduction in solids  
- Elasticity and solid mechanics  
- Diffusion and transport phenomena  
- Coupled physics and material behavior modeling  

---

### 5. Data-Driven Problems
- Problems involving **real and synthetic datasets**
- Integration of experimental or simulated data with physics constraints  
- Forward and inverse modeling cases
